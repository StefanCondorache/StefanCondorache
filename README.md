### Key Projects

#### **Cercetari-Operationale** | *University Project*
A collection of algorithms for Linear Programming and Game Theory.
* **Implementation:** Coded the **Primal Simplex Algorithm** using Python’s `fractions` module to ensure exact results instead of floating-point approximations.
* **Tools:** Includes a PySide6 GUI for inputting matrices and viewing step-by-step solutions.
* **Validation:** Tested against 25+ cases covering degeneracy and infeasibility.

#### **AI_Chess** | *Course Project (BME)*
A Deep Learning-based chess agent developed for an AI in Data Science course.
* **Architecture:** Uses a CNN with policy and value heads to evaluate board positions.
* **Training:** Trained on datasets of professional games combined with basic self-play reinforcement.
* **Features:** Real-time board visualization for move analysis.

#### **MyScripts** | *Personal Utilities*
A collection of Bash scripts for Linux system maintenance.
* **connect_bt:** A TUI-based Bluetooth manager that handles daemon hangs and identifies specific hardware (like Apple privacy protocols).
* **combo-toggle:** A utility to mitigate 2.4GHz Wi-Fi/Bluetooth interference by managing bands and BSSID locking.

#### **Company-Classifier** | *NLP Tool*
A Python utility designed to map company descriptions to insurance-specific taxonomies.
* **Logic:** Uses a mix of dictionary-based matching and vector-based search to categorize industry data from CSV files.

#### **ROCK-the-SHOP** | *C++ Academic Project*
A management system designed to demonstrate Object-Oriented Programming (OOP) principles.
* **Features:** Role-based logic for managers and operators, inventory tracking, and basic order processing.
